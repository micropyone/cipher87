    MkDocs

    • MkDocs
    • Material for MkDocs: mkdocs-material
    • highlight.js

    安裝 MkDocs

    • pip install mkdocs
    • pip install mkdocs-material

    操作指令

    • mkdocs new [dir-name] - 新建專案
    • mkdocs serve - 啟動網站伺服器。文件變更後即時重整更新頁面
    • mkdocs build - 建立靜態文件網站。可以部署至 GitHub
    • mkdocs -h - 指令說明
